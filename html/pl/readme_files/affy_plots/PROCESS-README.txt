Affymetrix Plots
Author: Nick Bild
Date: 2017-11-02
Version 1.0: Initial development.

Description
--------------------------------------------------------------------------------

The Affymetrix Plots app will generate QC images from raw text files generated by the Affymetrix Expression Console.

Script relationships:

affy_plots_run.pl
|
--affy_plots_pearsons_correlation_signal.pl
|
-- affy_plots_convert_rel_log_signal.pl
|
-- affy_plots_convert_signal_hist.pl

Dependencies
--------------------------------------------------------------------------------

Perl v5 is required. R 3.2.2 is required.  Required R libraries:

reshape2
dplyr
ggplots2

Software installed on marburg (192.168.0.14) in /home/act/software/nickbild/.

Usage Instructions
--------------------------------------------------------------------------------

* The pipeline is available on the ORB Intranet:

http://192.168.0.16/ -> Affymetrix -> Affymetrix Plots

The following questions will be asked:

* Enter a width (in pixels) for the output images:

* Enter a height (in pixels) for the output images:

* Select Pearson's Correlation data file:

This should be output directly from the Affymetrix Expression Console, e.g.: "Pearsons_Correlation_Signal_RMA.TXT".

* Select Relative Log Expression Boxplots data file:

This should be output directly from the Affymetrix Expression Console, e.g.: "Relative_Log_Expr_Signal_RMA_Boxplots.TXT".

* Select Signal Histogram data file:

This should be output directly from the Affymetrix Expression Console, e.g.: "Signal_Histogram_RMA.TXT".

* Select sampleinfo file (OPTIONAL):

The sampleinfo file must be a tab-delimited text file with hybridization IDs in the first column and a column named Customer ID.
Customer IDs will replace hybridization IDs in the generated images.
If not supplied, hybridization IDs will remain unchanged.

NOTE: All 3 data files are not required.  Plots will only be generated for the supplied data files.

Scripts
--------------------------------------------------------------------------------

* affy_plots_run.pl - Prepare data and generate QC images.

* affy_plots_pearsons_correlation_signal.pl - Prepare Pearson's Correlation data file.

* affy_plots_convert_rel_log_signal.pl - Prepare Relative Log Expression Boxplots data file.

* affy_plots_convert_signal_hist.pl - Prepare Signal Histogram data file.
